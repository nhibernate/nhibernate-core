Welcome to NHibernate!

NHibernate is a .NET based object persistence library for relational databases. NHibernate is a port of the excellent Java Hibernate relational persistence tool.

You can find tutorials and samples in the NHibernate Resources.

Latest Version

Details of the latest version of NHibernate can be found on SourceForge at http://sourceforge.net/projects/nhibernate/.

Documentation

All available resources on NHibernate can be found online: http://www.hibernate.org/22.html#A17.

The API documentation is in MSDN format; thanks to NDoc.

The help can also be integrated into Visual Studio.NET using the following steps:

  • Download and install H2Reg from http://helpware.net/mshelp2/h2reg.htm
  • Copy H2Reg.exe and H2Reg.ini from the H2Reg installation folder (typically C:\Program Files\HelpWare\H2Reg) to the NHibernate Help2 folder (e.g. C:\NHibernate\doc\en\Help2).
  • Register the help collection from the command line using the following command from the NHibernate Help2 folder.
H2Reg -r CmdFile=NHibernateCollection.ini
  • If you need to unregister the help collection, the command is as follows:
H2Reg -u CmdFile=NHibernateCollection.ini

The reference documentation for NHibernate can be found at http://www.hibernate.org/hib_docs/nhibernate/html/.
